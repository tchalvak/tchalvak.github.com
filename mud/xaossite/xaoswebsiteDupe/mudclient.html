Using other clients:

[Suggested connection method for new mudders]: (Requires no software installation or registration, but is initially a little more complex to set up)
-Gmud- "http://www.geneseo.edu/~ror1/e/Gmud.exe"; 350 KB Download; can run anywhere; from the web, from a floppy, or from the hard- drive; no registration necessary; has color, aliases, triggers, essentially all necessary features available.

Download GMUD from "http://www.geneseo.edu/~ror1/e/Gmud.exe".
Run the "Gmud.exe" program.
From the "File" option of the top menu, select "Connect".
Click the "Add" button and name the the world "Xaos", then hit "OK".
For "Host," type: xaos.ath.cx
For "Port", type: 5000
Click the "Connect" button to connect to Xaos.

Optional but suggested:
Give yourself more viewing room by clicking "View" in the upper menu and then unchecking "Connected World Window," "Macro Window," and "Status Bar".
Click on "Options" in the menu, then "Screen Wrap" and change the options to "Wrap to screen" and "word".
Click on "Options" again, then "Font" and change the font to "Courier New", the style to "Regular", and the size to "10".


[Best Long-Term mudding client]: (Newer versions eventually require a twenty dollar one-time payment, and software installation is necessary.)
-Zmud- "http://www.zuggsoft.com/zmud/zmudinfo.htm"; unable to run without being installed onto about 7 MB of hard-drive space; all the features you could ever need and much more; 30-day trial before it stops working unless you pay the 25$ registration fee.
