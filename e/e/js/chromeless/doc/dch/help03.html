MICROBIANS CHROMELESS WINDOWS


 MINI FAQ  
 

Hieronder volgend zijn een aantal van de meest gestelde vragen,die gesteld zijn door actuele gebruikers van MICROBIANS CHROMELESS WINDOWS.


HOE GEBRUIK IK VERSCHILLENDE PAGINA'S IN VERSCHILLENDE SCHERMEN?

Gebruik verschillende n waarden voor elke verschillende scherm in de functie:

openIT(u,W,H,X,Y,n)

Zodat de oproepen:

mywin1=openIT('p1.html',200,200,null,null,'mywin1')
mywin2=openIT('p2.html',200,200,null,null,'mywin2')

2 verschillende schermen moet openen.



HOE VERANDER IN DE PAGINA VAN DE TE OPENEN SCHERM?

Als voorbeeld wanneer de scherm was gecreëerd met:

mywin1=openIT('p1.html',200,200,null,null,'mywin1')

Kunnen we het veranderen naar p2.html pagina door hetvolgende te gebruiken:

mywin1.setURL('p2.html').



HOE OPEN IK EEN SCHERM VANUIT FLASH?

Om een scherm te openen vanuit flash, moet je een kijkje nemen naar het voorbeeld launch_flash5.html dat is bijgevoegd in de ZIP bestand en de launch.fla in de SWF map, ook te vinden in de ZIP bestand.

In de HTML bestand kan je zien dat de openIT is veranderd in openIT4FLASH. Omdat de Flash functie de javascript functie niet teruggeeft van het scherm, het is beter om de openIT4FLASH te gebruiken met een Flash programma. De actie script om in Flash een scherm te openen is:

getURL("javascript:openIT4FLASH('p1.html',400,400, null, null, 'mywin1')")

Voor Flash, werkt dit hetzelfde als de normale openIT functie.



HOE GEBRUIK IK SPECIALE KARAKTER CODES IN EEN TITEL?

Voor speciale karakter code's zoals Ñ of karakters met een teken, is het beter om codes te gebruiken als het volgende: &#64; of &Agrave;. Om een To find the individuele karakter code te vinden, bekijk dan alstublieft de informatie over de "ISO Latin-1 Character Set".


HOE SCROLL IK OF SCHAKEL IK DE SCROL FUNCTIE UIT?

Om de scrolfunctie uit te schalen in de MICROBIANS CHROMELESS WINDOWS compatibile schermen, gebruik je een style sheet zoals:

  <style>
    body { overflow:hidden }
  </style>

in de hoofd van elke html pagina of scroll=no in de body tag van de pagina's waar je geen scrolling wilt.

Maar om alle pagina's "niet scrollen" te maken, kan je ook de fSO ='scrolling=no noresize' variabele van de openIT functie aanpassen. Deze variabele bestuurt de weergave van de hoofd frame.

Er is een soortgelijke weergave voor niet MICROBIANS CHROMELESS WINDOWS compatibile schermen met de wNS variabele in de openIT functie. We kunnen bijvoorbeeld een scrolfunctie toevoegen aan Netscape met:

var wNS ='scrollbars=1'

binnen de openIT fuctie.
