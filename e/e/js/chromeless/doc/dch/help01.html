MICROBIANS CHROMELESS WINDOWS


 INTRODUCTIE  
 

MICROBIANS CHROMELESS WINDOWS is een set Javascript funties die het toestaan popup schermen te maken zonder de rand van de browser, dat de gebruiker toestaat zich beter te kunnen focussen op de inhoud van de site en, tegelijkertijd geeft het de volledige functionaliteit van een standaard scherm. Erbij komend is de schone design, het nut van Javascript effecten, en de flexibele inwendige karakteristiek van onze Chromeless Windows toevoegingen, zijn met niks op het web te vergelijken.

Onze Javascript functies zijn getest met Internet Explorer 4,5,5.5 en 6 op een Windows systeem, en ze openen een normale scherm in andere internet browsers, zoals Netscape (alle versies) en systemen (zoals Linux). Hoewel Internet Explorer de overmeesterende browser is, de voordelen van Chromeless Window deze feit overtreft. Extra volgen de niet-internet explorer schermen wel de opgegeven schermgrote op.
