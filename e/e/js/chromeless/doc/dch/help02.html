MICROBIANS CHROMELESS WINDOWS 3.5


 HOE TE GEBRUIKEN  
 

Om een Chromeless window te genereren, is de eerste stap in de Head tag van je HTML document de volgende code te plaatsen:


<script src="js/chromeless_35.js"></script>


Deze regel moet tussen de <HEAD> </HEAD>tags van de html document worden geplaatst die de Chromeless Window opent. Deze belangrijke voorzorg, zorgt ervoor dat alle nodige codes worden ingeladen alvorens dat ze nodig zijn. zodat ze direct klaarstaan wanneer ze opgeroepen worden voor gebruik.

Nadat die regel is geplaatst in de Head gedeelte, kan de "OpenIT" functie worden tewerk gesteld. Bekijk alstublieft het voorbeeld die in de zip file is bijgesloten.

Een "OpenIT" funtie is een in Javascript versimpelde geprogrammeerde service die een Chromeless Window zal openen.

Zoals elke code, een "OpenIT" functie lijkt moeilijk, totdat je ze zorgvuldig hebt doorgelezen, en ingebeeld hebt hoe veranderingen van elke functie een resultaat geven tegenover een verandering van een kenmerk van Chromeless Windows.

In de OpenIT functie zitten aan te passen velden, die Chromeless Window zijn flexibiliteit, schoonheid en flinkheid geven. Deze functies zijn:



KNOPPEN

cU   | gif voor sluiten in normale staat.
cO   | gif voor sluiten bij muisover.
cL   | gif voor laad indicator.
mU   | gif voor minimaliseren naar taakbalk in normale staat.
mO   | gif voor minimaliseren naar taakbalk bij muisover.
xU   | gif voor maximaliseren in normale staat.
xO   | gif voor maximaliseren bij muisover.
rU   | gif voor minimaliseren in normale staat.
rO   | gif voor minimaliseren bij muisover.


TITEL

tH   | titel voor de titelbalk in html formaat.
tW   | titel voor de taakbalk van Windows.


KLEUREN

wB   | Border kleur.
wBs  | Border kleur bij verslepen.
wBG  | Achtergrondkleur van titelbalk.
WBGs | Achtergrondkleur van titelbalk bij verslepen.


COMPATIBILITEIT

wNS  | Html parameters voor Netscape.
fSO  | Html parameters voor de hoofd frame inhoud.


EXTRA PARAMETERS

brd  | Extra border dikte.
max  | maximaliseren optie (true|false).
min  | Minimaliseren naar taakbalk optie (true|false).
res  | Scherm grote aanpasbaar (true|false).
tsz  | Hoogte van de titel balk.


OPENIT FUNCTION

Deze informatie is alleen voor de "OpenIT" functie die in de "test_html.html" voorbeeld zit Dat het gemakkelijk maakt de hoofd 'Chromeless' functie te gebruiken. Voor andere gebruiken of fx, is het beter je eigen "openIT" te maken.

openIT(u,W,H,X,Y,n,b,x,m,r)

u    | html pagina die we willen openen in het scherm.
W    | Breedte van de scherm.
H    | Hoogte van de scherm.
X    | Horizontale positie (null=center).
Y    | Verticale position (null=center).
n    | Scherm naam (handler).
b    | Extra border grote.
x    | Maximaliseren (true|false).
m    | Minimaliseren naar taakbalk (true|false).
r    | Scherm grote aanpasbaar (true|false).


OPENIT VOORBEELD

<a href="#" onclick="mywindow=openIT('mypage.html',200,200,null,null,'mywindowname')"> OPEN CHROMELESS</a>

Deze zou een scherm moeten openen met mypage.html als het onderwerp (inhoud) en 200x200 pixels als schermgrote, in het midden van het beeld. Controleer dat de pagina die geopent moet worden in de zelfde map zit als de pagina die het scherm opent, wanneer dit niet het geval is. Maak dan de URL compleet.
