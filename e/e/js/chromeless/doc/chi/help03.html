MICROBIANS CHROMELESS WINDOWS


 問答集  
 

這裡列舉使用者在使用 MICROBIANS CHROMELESS WINDOWS 時常見的一些問題。


在不同視窗開啟不同網頁

在下列功能中，給每一個不同的視窗使用不同的「n」變數：

openIT(u,W,H,X,Y,n)

如此呼叫：

mywin1=openIT('p1.html',200,200,null,null,'mywin1')
mywin2=openIT('p2.html',200,200,null,null,'mywin2')

將會開啟兩個視窗。



如何在開啟的視窗中換頁

例如有一個被開啟的視窗：

mywin1=openIT('p1.html',200,200,null,null,'mywin1')

我們可以使用下列語法以切換到 p2.html：

mywin1.setURL('p2.html').



如何利用FLASH來開啟一個視窗

利用flash開啟一個視窗，您可以觀看在壓縮包裡的範例launch_flash5.html以及 SWF 資料夾裡的launch.fla。

在 HTML 檔裡您可以看到我將 openIT 修改成 openIT4FLASH，這是因為這個新功能無法送回視窗的 javascript 程式碼，那樣會比較好使用 flash。 使用 flash 來開啟一個視窗的語法如下：

getURL("javascript:openIT4FLASH('p1.html',400,400, null, null, 'mywin1')")

可以像 openIT 功能一樣作用。



標題的特殊敘述代碼

特殊敘述代碼例如：N﹙N上面有兩撇﹚或其他敘述代碼，最好使用這樣的方法來表示：&#64; 或 &Agrave;。我建議您去尋找更多有關 "ISO Latin-1 Character Set" 的資訊。


使用捲軸與否

在 MICROBIANS CHROMELESS WINDOWS 相容瀏覽器中不使用捲軸，我們可以加入這樣一個 style 在每個不要有捲軸的網頁的 head 部分：

  <style>
    body { overflow:hidden }
  </style>

，或是加入 scroll=no 在網頁的 body 部分。

但若我們想要讓所有的網頁都不顯示捲軸，我們也可以用 fSO ='scrolling=no noresize' 這樣一個變數加入 openIT 功能中。這變數可以控制主要框架。

這裡有個類似的控制法是給非 MICROBIANS CHROMELESS WINDOWS 相容瀏覽器的，使用 wNS 這一個變數加入 openIT 功能裡。例如在 Netscape要顯示捲軸，可以利用：

var wNS ='scrollbars=1'

加入 openIT 功能裡。
