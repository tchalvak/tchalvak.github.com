MICROBIANS CHROMELESS WINDOWS


 INTRODUCTION  
 

MICROBIANS CHROMELESS WINDOWS is a set of JavaScript functions that allow the creation of popup windows without the chrome of the browser, allowing the user to better focus upon the content of a site and, at the same time, providing all of the functionality of a conventional window. Additionally, the clean design, utilization of JavaScript effects, and flexibility of the internal characteristics of our Chromeless Windows adds aesthetic unequaled on the web.

Our JavaScript functions have been tested on Internet Explorer 4,5,5.5 and 6 under the Windows platform, and they open a regular window in other browsers, such as Netscape (all versions) and platforms (such as Linux) However, since IE constitutes the overwhelming majority of browsers, the advantages of the Chromeless Window far outweigh this fact. Additionally, the popups from non-IE browsers follow generally the window size selected.
