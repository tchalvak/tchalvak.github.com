MICROBIANS CHROMELESS WINDOWS


 INTRODUCTION  
 

MICROBIANS CHROMELESS WINDOWS est un ensemble de fonctions Javascript qui permet la création de fenêtres popup sans les bordures grises habituelles du browser, ce qui permet à l'utilisateur de mieux se focaliser sur le contenu même du site, tout en gardant les mêmes fonctionnalités qu'une fenêtre conventionnelle. De plus, un design propre, l'utilisation d'effets Javascript, et la flexibilité des caractéristiques internes de Chromeless Windows ajoute une touche esthétique inégalée sur le Web.

 

Ces fonctions Javascript ont été testées sous Internet Explorer 4,5,5.5 et 6 sur plateforme Windows, et elles ouvrent une fenêtre popup standard sur les autres browsers, comme Netscape (toutes versions) ainsi que sur les autres plateformes (comme Linux). De toute manière, comme IE constitue l'écrasante majorité des browsers à l'heure actuelle, les avantages de Chromeless Windows sont la plupart du temps fonctionnels. De plus, les popups générés par des browsers autres qu'IE suivent généralement la taille spécifiée.
