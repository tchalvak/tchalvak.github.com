MICROBIANS CHROMELESS WINDOWS


 MINI FAQ  
 

Voici les questions les plus fréquemment posées par les utilisateurs de MICROBIANS CHROMELESS WINDOWS.


COMMENT PUIS-JE LANCER DIFFÉRENTES PAGES DANS DIFFÉRENTES FENÊTRES ?

Utilisez des valeurs n différentes pour chaque fenêtre dans la fonction:

openIT(u,W,H,X,Y,n)

Donc, ces deux appels:

mywin1=openIT('p1.html',200,200,null,null,'mywin1')
mywin2=openIT('p2.html',200,200,null,null,'mywin2')

devraient ouvrir deux fenêtres différentes.



COMMENT CHANGER DE PAGE DANS LE POPUP, À PARTIR DE LA FENÊTRE PRINCIPALE ?

Par exemple si le popup a été créé avec les paramètres suivants::

mywin1=openIT('p1.html',200,200,null,null,'mywin1')

nous pouvons changer son contenu vers p2.html en utilisant:

mywin1.setURL('p2.html').



COMMENT OUVRIR UNE FENÊTRE À PARTIR DE FLASH?

Pour ouvrir une fenêtre à partir de Flash, consultez l'exemplelaunch_flash5.htmlinclus dans le fichier ZIPainsi quelaunch.fla dans le repértoire SWF, lui aussi dans le fichier ZIP.

A la lecture du fichier HTML vous pouvez constater que la fonction openIT a été modifiée en openIT4FLASH. Comme la même fonction en Flash pur n'attribue pas la parenté de l'objet Javascript créé à la fenêtre principale, il est conseillé d'utiliser openIT4FLASH. L'Action Script en Flash pour appeler Chromeless sera donc:


getURL("javascript:openIT4FLASH('p1.html',400,400, null, null, 'mywin1')")

Sous Flash, les paramètres sont les mêmes, et l'usage de la fonction openIT est le même que ceux précédemment exposés.



COMMENT UTILISER DES CARACTÈRES SPÉCIAUX DANS UN TITRE?

Pour les caractères spéciaux comme Ñ ou les tildes et les accents , la meilleure manière de procéder consiste à utiliser le code du caractère comme par exemple &#64; or &Agrave;. Pour trouver un code de caractère paritculier, référez-vous à la documentation sur le "ISO Latin-1 Character Set".


COMMENT ÉLIMINER OU IMPOSER DES BARRES DE DÉFILEMENT?

Pour éliminer les barres de défilement, utilisez une feuille de style de ce type:

  <style>
    body { overflow:hidden }
  </style>

incluse dans le Head, ou scroll=no dans le marqueur body des pages dont vous voulez éliminer les barres de défilement.

Si vous voulez empêcher toutes les barres de défilement sur toutes les pages, introduisez la valeur fSO ='scrolling=no noresize'dans cette variable de la fonction openIT. Cette variable contrôle l'aspect du popup crée.

Il y a une autre variable similaire destinée aux browsers non compatibles avec MICROBIANS CHROMELESS WINDOWS. C'est la variablewNS dans la fonction openIT . Par exemple, on peut forcer l'apparition des barres de défilements sous Netscape avec:

var wNS ='scrollbars=1'

toujours à l'intérieur de la fonction openIT.
