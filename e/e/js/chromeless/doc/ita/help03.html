MICROBIANS CHROMELESS WINDOWS


 MINI FAQ  
 

Di seguito le domande più frequenti utilizzando la MICROBIANS CHROMELESS WINDOWS.


COME UTILIZZO PAGINE DIVERSE IN DIVERSE FINESTRE?

Usare differenti valori n per ogni finestra nella funzione:

openIT(u,W,H,X,Y,n)

In questo modo, la chiamata:

mywin1=openIT('p1.html',200,200,null,null,'mywin1')
mywin2=openIT('p2.html',200,200,null,null,'mywin2')

aprirà due diverse finestre.



COME POSSO CAMBIARE PAGINA DALLA FINESTRA D'APERTURA?

Per esempio se la finestra è stata creata con:

mywin1=openIT('p1.html',200,200,null,null,'mywin1')

possiamo aprire p2.html scrivendo:

mywin1.setURL('p2.html').



COME APRO UNA FINESTRA DA FLASH?

Lanciare l'esempio launch_flash5.html incluso nel file ZIP e il fla launch.fla nella cartella SWF sempre nel file ZIP.

Nel file HTML la funzione openIT è stata canbiata in openIT4FLASH, questo perchè Flash non restituisce l'oggetto JavaScript. E' quindi opportuno utilizzare openIT4FLASH con le applicazioni create con Flash. L' actionscript da utilizzare per aprire una finestra Chromeless è il seguente:

getURL("javascript:openIT4FLASH('p1.html',400,400, null, null, 'mywin1')")

l'effetto ottenuto è lo stesso della normale funzione openIT.



COME UTILIZZO I CARATTERI SPECIALI NEL TITOLO?

Per i caratteri speciali, come ad esempio Ñ o altri caratteri il sistema migliore è utilizzare la codifica: &#64; oppure &Agrave;. Per trovare il codice di un particolare carattere fare riferimento al "ISO Latin-1 Character Set".


COME CREO O DISABILITO LA FUNZIONE SCROLL?

Per eliminare lo scroll nella MICROBIANS CHROMELESS WINDOWS, compatibilità dei browsers permettendo, utilizzare un foglio di stile (style sheet) come:

  <style>
    body { overflow:hidden }
  </style>

nei tags head di ogni pagina html o scroll=no nei tags body delle pagine dove si vuole eliminare lo scroll.

Se si vuole invece eliminare lo scroll in tutte le pagine è possibile modificare la variabile fSO ='scrolling=no noresize' nella funzione openIT. Questa variabile controlla il "rendering" del frame principale.

Esiste un controllo simile per i browser non compatibili usando la variabile wNS sempre nella funzione openIT. Per esempio possiamo aggiungere lo scroll usando Netscape scrivendo:

var wNS ='scrollbars=1'

nella funzione openIT.
