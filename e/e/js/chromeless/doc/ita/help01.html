MICROBIANS CHROMELESS WINDOWS


 INTRODUZIONE  
 

MICROBIANS CHROMELESS WINDOWS è un set di funzioni JavaScript che permettono l'apertura di finestre window (popup) senza il classico bordo del browser, consentendo l'utente di visualizzare in maniera migliore il contenuto del sito e, allo stesso tempo, conservando tutte le funzionalità di una finestra windows convenzionale. In aggiunta il design pulito, l'utilizzo di effetti JavaScript, e la flessibilità delle caratteristiche interne della nostra Chromeless Windows, sono esteticamente senza eguali nel web.

Le nostre funzioni JavaScript sono state testate su piattaforma  Windows con Internet Explorer 4,5,5.5 e 6, e aprono una finestra regolare in altri browser, ad esempio in tutte le versioni di Netscape e altre piattaforme tipo Linux. In tutti i casi, poichè IE è sicuramente il browser più diffuso, i vantaggi dell'utilizzo della Chromeless Window rimangono inalterati. In aggiunta le finestre popup aperte con altri browser seguono generalmente le misure settate.
