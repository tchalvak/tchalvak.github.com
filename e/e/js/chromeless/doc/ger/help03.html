MICROBIANS CHROMELESS WINDOWS


 MINI FAQ  
 

Im Folgenden sehen sie die häufigsten Fragen von derzeitigen Nutzern des MICROBIANS CHROMELESS WINDOWS.


WIE LADE ICH VERSCHIEDENE SEITEN IN VERSCHIEDENE WINDOWS?

Benutzen Sie unterschiedliche n Werte für jedes neue Window innerhalb einer Funktion:

openIT(u,W,H,X,Y,n)

Die Aufrufe:

mywin1=openIT('p1.html',200,200,null,null,'mywin1')
mywin2=openIT('p2.html',200,200,null,null,'mywin2')

öffnen also zwei neue Fenster.



WIE ÄNDERE ICH DIE URL DER ÖFFNENDEN SEITE?

Wenn das Window mit folgendem geöffnet wurde:

mywin1=openIT('p1.html',200,200,null,null,'mywin1')

wechseln wir zur p2.html mit:

mywin1.setURL('p2.html').



WIE ÖFFNE ICH EIN FENSTER AUS FLASH HERAUS?

Scheuen Sie sich dazu das Beispiel launch_flash5.html im Zip an und auch die launch.fla im swf-Ordner.

Im HTML-File sehen sie, daß openIT zu openIT4FLASH wurde. Da Flash nicht den JavaScript Handler zurückgibt, ist es besser mit openIT4FLASH ein Fenster zu erstellen. Das Actionscript in Flash sollte dann so aussehen:

getURL("javascript:openIT4FLASH('p1.html',400,400, null, null, 'mywin1')")

Das arbeitet im Endeffekt genau so wie die normale openIT Funktion.



WIE KANN ICH SONDERZEICHEN IM TITEL VERWENDEN?

Für Sonderzeichen wie Ñ, Ä,Ö sollten sie die Codes: &#64; A&Uml oder &Agrave benutzen. Für die speziellen Codes suchen Sie bitte im "ISO Latin-1 Character Set".


WIE SCROLLE ICH ODER UNTERBINDE SCROLLING?

Um Scrolling in MICROBIANS CHROMELESS WINDOWS auszuschalten, benutzen Sie CSS:

  <style>
    body { overflow:hidden }
  </style>

im HEAD-Bereich jeder Seite oder scroll=no im BODY-Tag der Seite(n), die keinen Scrollbalken haben sollen..

Wenn keine der geöffneten Seiten einen Scrollbalken enthalten soll, benutzen Sie die fSO ='scrolling=no noresize' Variable der openIT Funktion. Diese variable kontrolliert das Rendering des zu öffnenden Fensters.

Es gibt eine ähnliche Funktion für nicht-MICROBIANS CHROMELESS WINDOWS-kompatible Browser in Form der wNS Variablen in der openIT Funktion. Zum Beispiel für Netscape:

var wNS ='scrollbars=1'

innerhalb der openIT Funktion.
