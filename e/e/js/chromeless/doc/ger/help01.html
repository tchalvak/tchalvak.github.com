MICROBIANS CHROMELESS WINDOWS


EINLEITUNG   
 

MICROBIANS CHROMELESS WINDOWS ist eine Sammlung von JavaScripts, die das kreieren von Pop-Up Windows ohne Browserrahmen erlauben, um den User auf den reinen Inhalt zu fokussieren ohne dabei an Funktionalität eines normalen Browserwindows zu verlieren. Zusätzlich sorgt das saubere Design, die Verwendung von JavaScript-Effekten und die Flexibilität der internen Einstellungsmöglichkeiten für eine unvergleichbare Ästhetik im Web.

Unsere JavaScript-Funktionen wurden für den Internet Explorer 4,5,5.5 und 6 unter Windows getestet. In allen anderen Browsern, wie Netscape (alle Versionen) und Betriebssystemen (so wie Linux) öffnen sie ein reguläres Browserwindow. Da der IE jedoch die absolute Mehrheit bildet, kommen die meisten User in den Genuss der Vorzüge unserer Chromeless Windows. Die Pop-Up Fenster aller anderen Browser folgen allerdings der angegebenen Größe in unserer Funktion, womit das Gesamtlayout nicht gestört wird.
