MICROBIANS CHROMELESS WINDOWS 3.5


 ANWENDUNG  
 

Um ein Chromeless Window zu erstellen, müssen sie grundsätzlich die folgende Zeile in den <Head>-Bereich ihres Dokuments einfügen.

<script src="js/chromeless_35.js"></script>


Diese Zeile muss zwischen den <HEAD> </HEAD>tags der HTML-Seite, die das Chromeless Window öffnen soll. Sie sichert, daß der gesamte, benötigte Code bereits geladen ist, wenn sie das Chromeless Window aufrufen.

Nachdem die zeile eingetragen wurde, kann nun die "openIT" Funktion verwendet werden. Bitte schauen Sie sich dazu das Beispiel im Zip-File an.

Eine "openIT" Funktion ist ein vereinfachter JavaScript Programmteil, der das Chromeless Window öffnet.

Wie immer bei Programmiercode, “openIT” sieht zum Anfang kompliziert aus, bis Sie sich die jeweiligen Features und Einstellungsmöglichkeiten verbildlicht haben und die Effekte auf das Chromeless Windows klar werden.

Innerhalb der openIT-Funktion sind definierbare Felder, die dem Chromeless Window sein Erscheiningsbild verleihen. Diese Funktionen sind:



GRAFIKEN

cU   | gif zum Schliessen im Normalmodus.
cO   | gif zum Schliessen im RollOver Modus.
cL   | gif für den Ladestatus.
mU   | gif zum Minimieren auf die Taskbar im Normalmodus.
mO   | gif zum Minimieren auf die Taskbar im RollOver Modus.
xU   | gif zum Maximieren im Normalmodus.
xO   | gif zum Maximieren im RollOver Modus.
rU   | gif zum Minimieren im Normalmodus.
rO   | gif zum Minimieren im RollOver Modus.


TITEL

tH   | Titel des Fensters im HTML-Format.
tW   | Titel des Fenster in der Taskbar.


FARBEN

wB   | Rahmenfarbe.
wBs  | Rahmenfarbe beim Fensterziehen.
wBG  | Hintergrundfarbe der Titelzeile des Fensters.
WBGs | Hintergrundfarbe der Titelzeile des Fensters beim Ziehen.


KOMPATIBILITÄT

wNS  | Html Parameter für Netscape.
fSO  | Html Parameter für main content Frame.


EXTRA PARAMETER

brd  | Extra Rahmengröße.
max  | Maximieren Option (true|false).
min  | Minimieren zur Taskbar (true|false).
res  | Resizable window (true|false).
tsz  | Höhe der Titelzeile.


OPENIT FUNKTION

Diese Informationen sind nur für die 'openIT' im enthaltenen 'test_html.html' Beispiel, die das Verwenden der Hauptfunktion erleichtern sollen. Für alle anderen Bereiche empfehlen wir eine eigene openIT-Funktion zu benutzen.

openIT(u,W,H,X,Y,n,b,x,m,r)

u    | HTML-Seite, die im Chromeless Window erscheinen soll..
W    | Breite des Windows..
H    | Höhe des Windows.
X    | Horizontale Position (null=center).
Y    | Vertikale Position (null=center).
n    | Window Name (handler).
b    | Extra Rahmengröße
x    | Maximieren (true|false).
m    | Minimieren zur Taskbar(true|false).
r    | Resize (true|false).

OPENIT BEISPIEL

<a href="#" onclick="mywindow=openIT('mypage.html',200,200,null,null,'mywindowname')"> OPEN CHROMELESS</a>

Das öffnet ein Chromeless Window mit dem Inhalt mypage.html und einer Größe von 200x200 Pixeln, in der Mitte des Bildschirms. Bitte stellen Sie sicher, daß sich die zu öffnende Seite im selben Verzeichnis befindet wie die aufrufende Seite; sollte das nicht möglich sein, geben sie bitte die komplette URL (inkl. http://) zur Inhaltsseite an.
