MICROBIANS CHROMELESS WINDOWS


 MINI FAQ  
 

Aquí se reflejan las dudas más comunes de los usuarios que han usado y usan las MICROBIANS CHROMELESS WINDOWS.


DIFERENTES PÁGINAS EN DIFERENTES VENTANAS

Usa diferentes n para cada ventana diferente en la función:

openIT(u,W,H,X,Y,n)

Así, las llamadas:

miwin1=openIT('p1.html',200,200,null,null,'miwin1')
miwin2=openIT('p2.html',200,200,null,null,'miwin2')

abrirían dos ventanas diferentes.



CAMBIAR DE PÁGINA DESDE LA VENTANA DE ORIGEN

Por ejemplo si se creó la ventana con:

miwin1=openIT('p1.html',200,200,null,null,'miwin1')

para cambiar a la página p2.html podemos usar:

miwin1.setURL('p2.html').



ABRIR UNA VENTANA DESDE FLASH

Para abrir una ventana ver el ejemplo incluido en el zip launch_flash5.html y el archivo launch.fla de la carpeta SWF. En el archivo html se puede apreciar que he cambiado openIT por openIT4FLASH y la función no retorna el objeto de la ventana. La acción de action script es de la forma del siguiente ejemplo:

getURL("javascript:openIT4FLASH('p1.html',400,400, null, null, 'miwin1')")

que funciona de forma similar a openIT mencionado anteriormente.



SÍMBOLOS ESPECIALES PARA EL TÍTULO

Con los caracteres especiales como la Ñ o caracteres acentuados se recomienda forzarlos en la forma &#64; o &Agrave;. Buscar información acerca de "ISO Latin-1 Character Set".


CONTENIDO CON y SIN SCROLL

Para eliminar el scroll podemos usar una hoja de estilo como:

  <style>
    body { overflow:hidden }
  </style>

o bien scroll=no en el tag body de las páginas que queremos que no tengan scroll.

Pero si lo que queremos controlar el scroll en general tenemos la opción de cambiar la variable fSO ='scrolling=no noresize' en la función openIT. Esta variable controla la forma en la que se renderiza el frame central de la ventana.

También tenemos control del scroll para navegadores no compatibles con la chromeless con la variable wNS en la función openIT. Por ejemplo activaríamos el scroll poniendo:

var wNS ='scrollbars=1'
